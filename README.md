# M Leying Notes

一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为通用型 HTML 内容存档使用。

## 项目简介

本项目旨在收集和整理一系列独立的 HTML 页面，这些页面可能包含笔记、工具、演示、参考文档或其他静态网页资源。  
所有页面均为自包含的 HTML 文件，可直接在浏览器中打开查看，无需后端支持。

## 目录结构

```
m-leying-notes/
├── pages/          # 存放所有 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/         # 公共资源文件（图片、样式、脚本等）
├── index.html      # 首页或导航页
└── README.md       # 本文件
```

- `pages/` 目录为页面主目录，每个 HTML 文件对应一个独立页面。
- `assets/` 目录用于存储页面间共享的资源文件，如图片、CSS、JavaScript 等。
- `index.html` 为可选入口文件，可用于展示页面列表或导航。

## 页面归档说明

- 每个 HTML 页面应尽量保持自包含，减少对外部资源的依赖。
- 页面命名应简洁明了，建议使用英文小写字母和连字符（如 `about-notes.html`）。
- 若页面涉及外部资源（如字体、库文件），建议托管至 `assets/` 目录或使用相对路径引用。
- 本仓库不保证页面内容的时效性或准确性，所有内容仅供存档参考。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面或改进现有页面。
- 提交前请确保页面无恶意内容、无外部跟踪脚本，并尽量保持文件体积较小。
- 若需要删除或更新页面，请通过 Issue 或 PR 说明理由。
- 本仓库由维护者不定期审核和处理提交内容。
